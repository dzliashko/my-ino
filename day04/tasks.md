# День 4. Методы

Сейчас у нас есть структура:

```
type Feed struct {
...
}
```

Мы можем "прикрепить" функцию к этой структуре.

Например:

```
func (f Feed) IsPopular() bool {
return f.Subscribers >= 1000
}
```

Использование:

```
feed.IsPopular()
```

### Задание №1

Удали функцию:

```
func feedRating(...)
```

и сделай метод:

```
func (f Feed) Rating() string
```

Чтобы работало:

```
fmt.Println(feedGo.Rating())
```

### Задание №2

Сделай метод:

```
func (f Feed) IsPopular() bool
```

Правила те же:

```
Subscribers >= 1000
```

### Задание №3

Сделай метод:

```
func (f Feed) Info() string
```

Чтобы можно было написать:

```
fmt.Print(feedGo.Info())
```

После этого

У тебя получится первый настоящий объект:

```
feed.Info()
feed.Rating()
feed.IsPopular()
```

Именно такой стиль потом будет использоваться в большинстве Go-проектов.
