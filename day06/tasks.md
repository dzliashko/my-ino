### Задание №1

Создай структуру Article.

### Задание №2

Создай две статьи.

Например:
```
Title:
Go 1.25 Released
```
и
```
Understanding Goroutines
```
Обе статьи принадлежат goFeed.

Используй:
```
PublishedAt: time.Now()
```

### Задание №3

Создай слайс:
```
articles := []Article{
    article1,
    article2,
}
```

### Задание №4

Напиши функцию
```
func printArticles(articles []Article)
```
Вывод должен быть примерно таким:
```
Feed: Go Blog
Title: Go 1.25 Released

Feed: Go Blog
Title: Understanding Goroutines
```

### Задание №5 (самое интересное)

Напиши функцию
```
func articlesByFeed(feed Feed, articles []Article) []Article
```
Она должна вернуть только статьи указанного канала.

Подсказка:
```
for _, article := range articles {
    if article.Feed.Name == feed.Name {
        ...
    }
}
```
Пока будем сравнивать по имени канала. Позже, когда появятся идентификаторы (ID), переделаем сравнение на них.

Что ты изучишь сегодня

После этого дня ты освоишь сразу несколько важных вещей:

 - импорт пакетов (time);
 - работу с time.Time;
 - вложенные структуры;
 - структуры как поля других структур;
 - фильтрацию слайсов по условию.

Это большой шаг вперёд: начиная с этого момента у нас уже будет не просто список каналов, а система из связанных сущностей — основа практически любого backend-приложения.

После выполнения, как обычно, присылай код на ревью.
