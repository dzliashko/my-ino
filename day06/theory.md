# День 6. Структуры внутри структур

До сих пор у нас был только канал.

Но RSS-канал существует ради статей.

Получается такая модель:
```
Feed
 ├── Go Blog
 │     ├── Article
 │     ├── Article
 │     └── Article
 │
 └── Hacker News
       ├── Article
       └── Article
```
Пока мы создавали только каналы.

Теперь научимся работать со статьями.

### Новая тема №1 — пакет time

Каждая статья имеет дату публикации.

В Go используется тип:
```
time.Time
```
Подключается пакет так:
```
import (
    "fmt"
    "time"
)
```
Создать текущее время:
```
published := time.Now()
```
Вывести:
```
fmt.Println(published)
```
### Новая тема №2 — структура Article

Создай новую структуру.
```
type Article struct {
    Title       string
    Description string
    Link        string
    PublishedAt time.Time
}
```
Обрати внимание:
```
PublishedAt time.Time
```
Это первое поле, которое имеет не встроенный тип, а тип из другого пакета.

### Новая тема №3 — вложенные структуры

Самый интересный момент.

Есть два варианта.

**Вариант 1**

У статьи хранится только название канала.
```
FeedName string
```

**Вариант 2**

У статьи хранится весь канал.
```
Feed Feed
```
То есть структура содержит другую структуру.

Именно этот вариант мы будем использовать.

Получится:
```
type Article struct {
    Title       string
    Description string
    Link        string
    PublishedAt time.Time
    Feed        Feed
}
```
Теперь статья знает, к какому каналу относится.
